FunkyBot

FunkyBot is a general purpose chatbot for Discord. It contains several commands ranging from useful to silly. It was built by leveraging Discord.py and was intended for personal use but can be freely used on any server. New functionality is added as I get ideas or suggestions, and I often use FunkyBot as a way to experiment with new ideas.

Some commands available to use with FunkyBot include:

  • !react/!shibe - Post a random reaction image or picture of a shiba inu to chat. No images are supplied in the project repository and must be added by the user running the program.
  • !magic - Search for a Magic: the Gathering card by name or randomly. This command uses the Scryfall API to access card data.
  • !remind/!announce - Set a reminder to alert either you or the entire server (if you are an admin) after a specified time. Time can be entered either as a duration of time from after the command is used, or set to a specific date.
