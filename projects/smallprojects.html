Color Picker/Color Blender

A pair of small Android apps written in Kotlin. Color Picker allows the user to use red, green, and blue sliders to manually select a color while Color Blender can be used to import colors and mix two of them proportionally.

Collection Organizer

An Android app written in Kotlin meant to catalog and organize collections of items. The user can set custom categories and pictures for each item as well as rating them out of five stars, to allow the app to be used as a catalog for anything. It leverages the phone's camera app to take pictures and store them for direct use.

MtG Promo Finder

A JavaScript tool that allows users to search for Magic: the Gathering cards and supplies them with a list of promotional and premium printings. It uses the Scryfall API to find cards and can be customized to return only specific types of printings.
