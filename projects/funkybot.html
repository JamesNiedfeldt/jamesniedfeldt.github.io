FunkyBot

FunkyBot is a general purpose chatbot for Discord. It contains several commands ranging from useful to silly. It was built by using Discord.py and was intended for personal use but can be freely used on any server. New functionality is added as I get ideas or suggestions, and I often use FunkyBot as a way to experiment with new ideas.

Project repository

FunkyBot's commands include:

  • !hello - Get version and uptime information.
  • !help [[X]] - Get a list of commands or help with a specific command.
  • !announce [[X|...]] - Set a reminder that tags everyone on the server after a set time. Usable only by administrators.
  • !binary [[X]] - Convert an integer to binary.
  • !hex [[X]] - Convert an integer to hexadecimal.
  • !magic [[X|...]] - Search Scryfall for Magic the Gathering cards.
  • !poll [[X|Y|...]] - Create a poll with options to be voted on by server members.
  • !remind [[X]] - Set a reminder that tags you after a set time.
  • !roll [[X]] - Roll an X-sided die.
  • !wiki [[X]] - Search Wikipedia for an article.
  • !ask - Answer a yes or no question.
  • !choose [[X|Y|...]] - Choose between multiple options.
  • !cute - Send a random cute animal picture.
  • !joke - Set a random one-liner.
  • !react - Send a random reaction image.
  • !rate [[X]] - Rate something on a scale from 1-10.
