BlueJ Quality Assessment Tools

Quality Assessment Tools is an extension for BlueJ that checks the quality of the users' Java code against a set of tools. Its current iteration features Checkstyle and PMD checks, but the extension allows for future developers to add additional tools.

This project was worked on for a class project as a team and based off of foundational work found here. Our contributions involved expanding on the original BlueJ Checkstyle Extension to allow for PMD as well as restructuring the existing package to allow for ease of further development and implementation of additional tools.

Users can edit preferences for each quality assessment tool, such as what properties for Checkstyle and PMD to evaluate. PMD checks are run when the code is compiled, and works in the background to produce a report so that the flow of work isn't hindered. For future developers, the existing Checkstyle for BlueJ extension code was reworked and components were decoupled to allow for greater ease of expansion.
