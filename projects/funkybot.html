FunkyBot

FunkyBot is a general purpose chatbot for Discord. It contains several commands ranging from useful to silly. It was built by using Discord.py and was intended for personal use but can be freely used on any server. New functionality is added as I get ideas or suggestions, and I often use FunkyBot as a way to experiment with new ideas.

Some commands available to use with FunkyBot include:

  • !poll - Create a poll with a custom question and answers from which server users can pick via message reactions. It will run for an hour then automatically display the results.
  • !magic - Search for a Magic: the Gathering card by name or randomly. This command uses the Scryfall API to access card data.
  • !remind/!announce - Set a reminder to alert either you or the entire server (if you are an admin) after a specified time. Time can be entered either as a duration of time from after the command is used, or set to a specific date.
